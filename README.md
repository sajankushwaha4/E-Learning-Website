# 📚 E-Learning Website

A modern and responsive E-Learning website built using HTML5, CSS3, JavaScript, and Bootstrap. This project focuses on creating a clean, user-friendly interface for an online learning platform.

## 🚀 Features

- Responsive Design
- Modern UI/UX
- Home Page
- About Page
- Courses Page
- Contact Page
- Mobile Friendly
- Cross Browser Compatible
- Smooth Navigation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Font Awesome
- Google Fonts

## 📂 Folder Structure

```
E-Learning/
│── index.html
│── css/
│── js/
│── images/
└── assets/
```

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/E-Learning.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
